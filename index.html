About

Hey, my name is Yehuda! I'm a former Photographer turned Software Engineer. I love learning new skills and challenging myself creatively and technically to learn something new each day and wanted to pursue that feeling in my professional career. Whenever I wanted something or needed something done, I would take the time to learn all about it and how to do it myself. When I wanted a new computer, I took the time and effort to learn all about what went into building a computer and what parts I would need. Then I built it myself! I took a dive into the software engineering world when I was accepted into App Academy, a 1000+ hour intensive bootcamp, where I learned technologies such as React, Redux, JavaScript, Ruby, Ruby on Rails, MongoDB, Express.js, Node.js, HTML, CSS, Webpack, SWL, PostgreSQL, jQuery, Git, Heroku and AWS.

In everything I build, I strive to put myself in a user's point of view and develop an intuitive user interface

Some of my favorite things to do when i'm not programming include nature and landscape photography.
